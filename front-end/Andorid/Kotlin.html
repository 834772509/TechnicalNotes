# 数据类型

  • Int : 整形
  • Long : 长整型
  • Short 短整形
  • Float : 单精度浮点型
  • Double : 双精度浮点型
  • Boolean : 布尔型
  • Char : 字符型
  • String : 字符串
  • Byte : 字节型

类型转换：

  • toByte()：转换为 Byte 类型
  • toShort()：转换为 Short 类型
  • toInt()：转换为 Int 类型
  • toLong()：转换为 Long 类型
  • toFloat()：转换为 Float 类型
  • toDouble()：转换为 Double 类型
  • toChar()：转换为 Char 类型

# 修饰符

  • public : 所有类可见（默认）
  • private : 当前类可见

  • internal : 同一模块中的类可见
