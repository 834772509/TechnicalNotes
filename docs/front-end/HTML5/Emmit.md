# Emmit

Emmet 可以用缩写来动态生成我们所需要生成出我们所需要的样式，在短时间做出来一个模板，然后使用，是一个能够大幅度提高效率的工具

## 快速输入

- .类名 : `<div class="类名"></div>`
- div : `<div></div>`

## 语法

### 层级结构

`ul>li`

```HTML
<ul>
  <li></li>
</ul>
```

### 同级结构

`div+h1`

```HTML
<div></div>
<h1></h1>
```

### 批量复制

`ul>li*3`

```HTML
<ul>
  <li></li>
  <li></li>
  <li></li>
</ul>
```

### 内容填充

`a{首页}`

```html
<a>首页</a>
```

### 内容自增

`ul>li{第$$条}*3`

```html
<ul>
  <li>第01条项目</li>
  <li>第02条项目</li>
  <li>第03条项目</li>
</ul>
```
