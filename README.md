# 杉-技术笔记

## 介绍

使用 VuePress 搭建的技术笔记

## 软件架构

使用 Vuepress 生成

## 安装教程

1. 安装[Node.js](https://nodejs.org/zh-cn/)
2. 配置镜像: `npm config set registry https://registry.npm.taobao.org`
3. 安装依赖: `npm install`

## 使用说明

- 本地预览: `npm run dev`
- 编译打包: `npm run build`
